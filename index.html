Explain, in a couple of paragraphs, what you think 2 major challenges around DNS configuration are for less-technical internet end-users

The first major challenge in my opinion for DNS configuration has to do with configuring your domain name. For people that have never set up a host name they would see the list of records such as A, AAAA, CNAME, etc. and right then and there they would not know how to proceed because they do not know which ones to fill out or if you should all out of them. An example would be if you wanted to use both www and a non-www domain, then you would need to add both those records. One to A type and the other to CNAME.

The second challenge is when they finally do put in a host name in the correct record then the user might be wondering why it is taking so long for their site to appear because they are getting a 503 error but sometimes the propagation takes up to 24 hours so they may be trying different records without realizing they actually wrote it correctly. You may also need to wait a long time when you change the name servers.
